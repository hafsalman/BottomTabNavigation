# 📱 Bottom Tab Navigation App

A mobile application built using **React Native with Expo** that demonstrates the implementation of Bottom Tab Navigation using React Navigation.

The app allows users to switch between multiple screens using a bottom tab bar interface, commonly used in modern mobile applications.

---

## Features

- Bottom Tab Navigation
- Seamless screen switching
- Custom tab icons
- Responsive and clean UI
- Smooth navigation transitions

---

## Tech Stack

- React Native
- Expo
- JavaScript (ES6)
- React Navigation
- Node.js
- Git & GitHub

## Dependencies
``` bash
npm install @react-navigation/native
npm install @react-navigation/bottom-tabs
npm install react-native-screens react-native-safe-area-context
npm install react-native-vector-icons

npx expo install react-native-vector-icons

npx expo install react-native-screens@~4.16.0
```

## Installation & Setup

### 1️⃣ Clone the Repository

``` bash
git clone https://github.com/hafsalman/BottomTabNavigation
```

### 2️⃣ Install Dependencies 

``` bash
npx create-expo-app BottomTabNav --template expo-template-blank
```

### 3️⃣ Navigate to Project Folder

``` bash
cd SignIn
```
(Replace the default App.js with the cloned App.js and add the rest of the folders.)

### 4️⃣ Install Dependencies

``` bash
npm install @react-navigation/native
npm install @react-navigation/bottom-tabs
npm install react-native-screens react-native-safe-area-context
npm install react-native-vector-icons

npx expo install react-native-vector-icons

npx expo install react-native-screens@~4.16.0
```

### 5️⃣ Start the Development Server

``` bash
npx expo start
```
